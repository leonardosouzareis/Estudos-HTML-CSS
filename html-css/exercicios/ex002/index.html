Usando <p> e <br>

Você pode escrever o que quiser, basta por entre o par de tags <p>. Mesmo apertando ENTER antes do início desta frase, tudo continua na mesma linha, pois esta frase está entre o mesmo par de tags <p> que a primeira frase.
